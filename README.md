# shopping-cart

Minimal shopping cart to demo on https://salus.cloud


## Deployment Instructions

The Shopping Cart is deployable

- As a Javascript Frontend app for support platforms
- As a webapp by leveraging the provided Dockerfile.

### Port

If deployed as a webapp, the app is exposed on port `8080` .

### Environment Variables

- `VITE_GO_API_URL` - URL for the shopping cart API, without the trailing slash. e.g. `http://localhost:8080`.
